# MiniCodeAgent Review Pack

## Project pitch

This directory collects benchmark and review artifacts used to evaluate MiniCodeAgent runs.

## Architecture map

CLI -> Runtime -> Context Manager -> Model Client -> Tool Executor -> Trace and Report.

## Benchmark evidence

Benchmark artifacts are generated by the evaluator and stored as JSON reports for repeatable review.

## Sample run artifact list

- task_state.json
- trace.jsonl
- report.json
